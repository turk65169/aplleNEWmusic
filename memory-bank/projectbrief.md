# Project Brief: Apple Music Bot

## Foundation
Apple Music Bot is a high-performance Telegram music bot built with Python and the Kurigram library. It provides high-quality audio and video streaming in Telegram voice chats.

## Core Requirements
- **High Quality Streaming**: Provide clear audio and video playback.
- **YouTube Integration**: Search and play music directly from YouTube.
- **Multi-language Support**: Interface available in multiple languages (es, zh, de, ru, hi, fr, my, pa, ja, en, ar, pt).
- **Admin Controls**: Robust management features for group administrators.
- **Queue Management**: Intelligent music queuing system.
- **UX Excellence**: Premium UI with animated emojis and colored buttons (Kurigram features).

## Project Scope
- Development and maintenance of the bot core (`KumsalTR`).
- Integration with YouTube via `yt-dlp`.
- Voice chat management using `py-tgcalls`.
- Database storage with MongoDB.
- Localization of all user-facing strings.
