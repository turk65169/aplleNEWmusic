<h3 align="center">
    ─「 📜 KOMUTLAR / YARDIM 」─
</h3>

### 🎵 Müzik Komutları

- `/oynat` → Şarkı çalar  
- `/voynat` → Video oynatır  
- `/duraklat` → Müziği duraklatır  
- `/devam` → Müziği devam ettirir  
- `/atla` → Sıradaki müziğe geçer  
- `/durdur` → Müziği tamamen durdurur  
- `/sira` → Müzik sırasını gösterir  

### 🔎 Arama Komutları

- `/sarki` → Şarkı indirir  
- `/video` → Video indirir  
- `/soz` → Şarkı sözlerini gösterir  

### ⚙️ Bot Komutları

- `/ping` → Bot gecikmesini gösterir  
- `/istatistik` → Bot istatistiklerini gösterir  
- `/yardim` → Yardım menüsünü açar  
- `/baslat` → Botu başlatır  

### 👑 Admin Komutları

- `/sustur` → Botu susturur  
- `/ac` → Botu tekrar açar  
- `/bitir` → Yayını kapatır  
- `/atla` → Şarkıyı atlar
